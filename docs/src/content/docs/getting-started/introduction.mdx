NGFW.sh is a cloud-managed next-generation firewall platform that brings enterprise-grade security to home and small business networks.

## What is NGFW.sh?

NGFW.sh combines powerful on-device firewall capabilities with a cloud management portal, giving you:

- **Real-time traffic monitoring** - See what's happening on your network
- **Advanced threat protection** - IDS/IPS, malware blocking, and DNS filtering
- **Easy VPN access** - WireGuard VPN server with QR code provisioning
- **Fleet management** - Manage multiple devices from a single dashboard

## Architecture

NGFW.sh consists of three main components:

1. **Router Agent** - Runs on your router hardware, executing firewall rules and reporting metrics
2. **Cloud API** - Hosted on Cloudflare Workers, handles authentication and configuration storage
3. **Web Portal** - React-based dashboard for configuration and monitoring
