NGFW.sh includes an Intrusion Detection and Prevention System to protect against network threats.

## Modes

- **Detect** - Log threats without blocking
- **Prevent** - Actively block detected threats

## Rule Categories

Enable or disable rule categories based on your needs:
- Malware
- Exploits
- Scanning
- Policy violations

## Alerts

View threat alerts in real-time and configure notifications.

See the [API Reference](/api/overview) for programmatic configuration.
