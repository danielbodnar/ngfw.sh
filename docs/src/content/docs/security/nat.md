Network Address Translation (NAT) allows devices on your private network to access the internet.

## Port Forwarding

Forward external ports to internal services:

- Web servers
- Game servers
- Remote access

## UPnP

Automatic port forwarding for compatible applications. Can be enabled or disabled for security.

See the [API Reference](/api/overview) for programmatic configuration.
