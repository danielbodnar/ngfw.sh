The built-in DHCP server automatically assigns IP addresses to devices on your network.

## Configuration

- IP range
- Lease time
- DNS servers
- Gateway

## Static Reservations

Assign fixed IP addresses to specific devices by MAC address.

## Lease Management

View and manage active DHCP leases from the dashboard.

See the [API Reference](/api/overview) for programmatic configuration.
