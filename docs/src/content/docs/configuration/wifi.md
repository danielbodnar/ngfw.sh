Set up and manage your wireless networks with NGFW.sh.

## Radio Configuration

Configure radio settings like channel, width, and transmit power for optimal performance.

## Creating Networks

Create multiple SSIDs for different purposes:

- **Primary network** - Main home/office WiFi
- **Guest network** - Isolated guest access
- **IoT network** - Smart device network

## Security

NGFW.sh supports:
- WPA3 (recommended)
- WPA2/WPA3 mixed mode
- WPA2

See the [API Reference](/api/overview) for programmatic configuration.
