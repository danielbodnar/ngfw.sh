NGFW.sh includes a built-in WireGuard VPN server for secure remote access to your network.

## Setup

1. Enable the VPN server
2. Configure the listen port and address range
3. Create peer configurations

## Adding Peers

Create configurations for each device:
- Mobile phones
- Laptops
- Other routers

Use the QR code feature for easy mobile setup.

## Security

WireGuard provides modern, fast, and secure VPN connections using state-of-the-art cryptography.

See the [API Reference](/api/overview) for programmatic configuration.
