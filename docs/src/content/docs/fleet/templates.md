title: Templates
description: Apply configuration templates to multiple devices
---

Templates allow you to define configurations once and apply them to multiple devices.

## Creating Templates

Define common configurations:
- Firewall rules
- DNS settings
- WiFi networks

## Applying Templates

Select devices and apply templates in bulk.

## Template Variables

Use variables for device-specific values like hostnames and IP addresses.

See the [API Reference](/api/overview) for programmatic configuration.
