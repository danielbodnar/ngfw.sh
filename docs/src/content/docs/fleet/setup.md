Fleet management allows you to manage multiple NGFW.sh devices from a single dashboard.

## Adding Devices

Register devices from the Fleet section of the dashboard.

## Centralized Monitoring

View status, alerts, and metrics for all devices at a glance.

## Remote Commands

Send commands to devices:
- Reboot
- Configuration push
- Firmware upgrade

See the [API Reference](/api/overview) for programmatic configuration.
